## 2025/9/29(月曜日)
Rubyの例外処理の書き方と正規表現について触れた。  
例外処理はraiseで強制的に例外を発生させることができる。  
例外処理は、begin〜rescue〜endの形で書く。  
正規表現は文字列パターンを記述するための記法で、
メソッドと組み合わせることで、 
文字列の検索・置換などができるようになる。  
クラス以降のことに関しては、
実際のWebアプリ制作で数をこなすしかないと感じた。
開発に必要なコマンドについては、  
Rubyの学習で散々やったので軽めに。
