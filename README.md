## 2025/9/22(月曜日)
Rubyの配列を扱うメソッドについて学習した。  
とにかく扱うメソッドの数が多過ぎた。  
```
unique_array = array.uniq
sorted_array = unique_array.sort
```
ここは以下のように省略できる。  
```
array.uniq.sort
```
